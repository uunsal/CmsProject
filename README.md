# CmsProject
Personal website management application


****Features:**

- 	Multi-user support
- 	Theme change
- 	Customized css
- 	Changing the menu order
- 	Flexible management panel
- 	Rich text editor support
- 	Angular JS engine
- 	Compatible web pages with Bootstrap
- 	Picture loading feature
- 	Platform independent operation
-	Spring security
